# SimpleQuestPlates Roadmap

This project is feature-complete for the current release line.
Roadmap items are focused on maintenance, compatibility, and quality.

## Current State

- Addon functionality is stable for quest-nameplate workflows.
- Options UI has dedicated tabs for global, kill, loot, and percent behavior.
- Release workflow is automated through GitHub tag pushes.

## Next Priorities

1. Compatibility maintenance
- Track WoW API changes across supported client branches.
- Keep compatibility helpers minimal and tested in-game after major patches.

2. Localization quality
- Fill translation gaps for newer command and options strings.
- Add reviewer pass for non-English locale consistency.

3. Performance verification
- Continue validating behavior in crowded nameplate scenarios.
- Review event/update paths after significant feature changes.

4. Release hygiene
- Keep changelog files single-version and accurate.
- Keep repository docs aligned with runtime architecture.

## Deferred

- New feature work beyond core quest plate flow will be considered only from user feedback and proven demand.
