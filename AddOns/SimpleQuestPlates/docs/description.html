# 🎯 RGX | Simple Quest Plates! ## ![](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Simple Quest Plates!** displays quest progress icons directly on enemy nameplates in World of Warcraft, making it instantly clear which enemies you need to kill for your active quests. Fully customizable with multiple icon styles, sizes, and positioning options. ![RealmGX Kiwi](https://github.com/donniedice/SimpleQuestPlates/blob/main/images/kiwi.gif?raw=true) **The Kiwi Says:** "Quest progress at a glance! Bwwiiiee." *** ## ✨ Features 🎯 Quest Progress Icons - See which enemies count for your active quests — directly on their nameplates! 🎨 Extensive Customization - Multiple icon styles, sizes, and positioning options to match your UI. ⚡ Lightweight - Minimal performance impact with efficient event-driven updates. 🌍 Localization - Supports 11 languages including English, Russian, German, French, Spanish, Korean, and more! 🔄 Real-Time Updates - Icons update instantly as you accept, progress, and complete quests. 💾 Persistent Settings - Your customization settings are saved automatically between sessions. 🎮 Retail Focused - Built and optimized for WoW Retail Midnight for maximum compatibility. 🛡️ Error Protected - Robust error handling for maximum stability during your questing sessions. **Note:** Works with Blizzard's default nameplates and most nameplate addons. *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/sqp # Opens the options panel
/sqp help # Displays help information
/sqp enable # Enable the addon
/sqp disable # Disable the addon
/sqp toggle # Toggle the addon on/off
/sqp reload # Reload addon settings
