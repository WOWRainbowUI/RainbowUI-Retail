# /images

This directory contains the image assets used by the Simple Quest Plates addon.

## Image Files

| File | Description |
|---|---|
| `icon.tga` | The main addon icon displayed in-game and in addon managers. This is a 32x32 Targa file. |
| `logo.png` | The addon's logo, used in the options panel and for promotional materials. |
| `in-game-options.png` | A screenshot of the in-game options panel, used for the addon's webpage. |
| `kiwi.gif` | A placeholder or decorative image, potentially for the RGX community tab. |
