> **Version 5.5 - Travel Improvements & Anchor Fixes**
>
> This update focuses on **Travel** module improvements across multiple game versions, with a new Retail hearthstone addition and cleaner Classic options.
> It also fixes several anchor-related issues that could cause the **Travel** and **Gold** modules to disappear.

### _Global :_

- 🐞 _**BUGFIX** -_ Fixed an issue that sometimes caused the **Travel** module to disappear when changing zones.
- 🐞 _**BUGFIX** -_ Fixed an issue that sometimes caused the **Gold** module to disappear.
- 🐞 _**BUGFIX** -_ Fixed an issue where checking **Hide Hearthstone Button** in the settings could leave the **Travel** module empty and cause other anchored modules to disappear.

### _Retail :_

- 🆕 _**NEW** -_ Added **Lightcalled Hearthstone** to the **Travel** module hearthstone list.

### _Classic :_

- 🛠️ _**IMPROVEMENT** -_ The secondary teleports should no longer appear in the **Travel** module options.
