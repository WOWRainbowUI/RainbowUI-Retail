# XIV_Databar Continued

## [4.8](https://github.com/ZelionGG/XIV_Databar-Continued/releases/tag/4.8) (2026-03-09)

[Full Changelog](https://github.com/ZelionGG/XIV_Databar-Continued/compare/v4.7.1...v4.8) [Previous Releases](https://github.com/ZelionGG/XIV_Databar-Continued/releases)

### _Global :_

- 🆕 _**NEW** -_ It is now possible to right-click on the **Currency** button to open the **Currency** module options.
- 🆕 _**NEW** -_ The **Game Menu** button now shows a tooltip that explains the different click actions available.
- 🛠️ _**IMPROVEMENT** -_ The **Reputation** module tooltip now uses the Blizzard's global string for the Left-Button description.
- 🐞 _**BUGFIX** -_ Fixed LUA errors caused by the Guild **Message of the Day** in the **Micromenu** module.
- 🐞 _**BUGFIX** -_ Fixed LUA errors caused by the **Currency** module.

### _Retail :_

- 🔥 _**IMPORTANT** -_ The War Within Season 3 is now flagged as over. The **Mythic+ Teleports** button will show all available teleports until the next season starts, and a new **Hide button during off-season** option is available in the **Travel** module to allow hiding the button instead.
- 🆕 _**NEW** -_ A new **Show unlearned teleports** option is now available (enabled by default) in the **Travel** module to list unlearned **Mythic+ Teleports**.
