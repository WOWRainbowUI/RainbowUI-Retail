# v1.7.0 - 2026-05-02

## Changes

- Rewrote `UI:CreateSlider` to use a custom track-style slider with RGX brand colors (primary fill/thumb, border track) instead of Blizzard's `OptionsSliderTemplate`.
- New slider supports click, drag, scroll wheel, hover value label, and built-in reset button.
- Fill/thumb positions use percentage-based layout with a one-shot `OnUpdate` handler to fix stale positioning when the track hasn't rendered yet.
- Added `UI:CreateVolumeSlider` for discrete 3-position (Low/Medium/High) volume control with the same custom track styling.
- Volume slider label appears below on hover; thumb positions at 15% (low), 50% (medium), 85% (high).
- Both slider types use `Design:Unpack("primary")` and `Design:Unpack("border")` for brand-consistent theming.

## Fixes

- Fixed reset button vertical alignment on the new slider (moved from -16 to -10 offset).
