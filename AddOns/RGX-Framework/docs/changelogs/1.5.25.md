# v1.5.25 - 2025-04-29

## Diagnostics

Added in-game `/rgx ddtest` command to diagnose dropdown rendering issues:
- Opens a test panel with two dropdowns
- Simple canary: Alpha/Beta/Gamma items
- Font canary: First 10 available fonts
- Click items to see selection logging in chat
- Use `/rgx ddhide` to close the panel
- Use `/rgx debug` to toggle debug logging

This helps isolate whether dropdowns fail to render text, rows, or both, without involving SQP/PB2.

> **Note:** `/rgx ddtest` and `/rgx ddhide` were temporary diagnostic commands removed in v1.5.26 once the dropdown rendering issue was resolved. They are no longer available.

## How to Test

In-game console:
```
/rgx ddtest -- Open test panel
/rgx ddhide -- Hide test panel
/rgx debug -- Toggle debug logging
```

## Expected Observations

**If Alpha/Beta/Gamma display text:**
- RGXDropdowns core rendering works
- If fonts dropdown is blank: problem is font item structure or font styling

**If Alpha/Beta/Gamma also blank:**
- RGXDropdowns/MenuUtil path or template is broken on Retail 12.x
- Row generation or text rendering is failing

## Next Steps

Test results will determine whether to fix RGXDropdowns or font item conversion.
