# Changelog

## Release

- Version bumped to `v12.016`.
- General tab Blizzard CT controls were stabilized:
- `Enable Blizzard CT In Group` now controls Blizzard CT only while grouped.
- `Disable Blizzard CT While Solo` now controls Blizzard CT only while solo.
- Fixed the conflict where enabling group Blizzard CT could incorrectly force solo Blizzard CT off.
- Removed extra Blizzard group child toggles and simplified behavior to the single group toggle.
- Added Escape-key close support for MSBT Options window.
